# AIlowerlimb

AIlowerlimb is a Windows-based AI-assisted software application for
lower-limb radiographic measurement.

## Main Functions

- DICOM, PNG, JPEG, and TIFF image loading
- YOLO11-pose keypoint detection
- Automated lower-limb alignment measurements
- Manual correction of 18 anatomical keypoints
- Recalculation of final measurements
- JSON and annotated-image export
